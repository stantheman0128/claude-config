# Claude Code Configuration

我的 Claude Code 编辑器配置和工作流指南，用于在多台电脑间同步设置。

## 文件说明

- **settings.json**: Claude Code 的配置文件
  - 启用的插件设置
  - 权限配置
  - 模型和更新频道设置

- **CLAUDE.md**: 工作流和项目管理指南
  - 计划模式和子代理策略
  - 自我改进循环
  - 核心原则和最佳实践

## 使用方法

将这些文件复制到 `~/.claude/` 目录（Linux/macOS）或 `C:\Users\<username>\.claude\` 目录（Windows）。

## 同步

在任何一台电脑上更新设置后，提交并推送到这个仓库，其他电脑就可以拉取最新配置。
