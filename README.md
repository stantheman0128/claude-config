# Claude Code Configuration

我的 Claude Code 編輯器配置和工作流程指南，用於在多台電腦間同步設定。

## 檔案說明

- **settings.json**: Claude Code 的配置檔案
  - 啟用的插件設定
  - 權限配置
  - 模型和更新頻道設定

- **CLAUDE.md**: 工作流程和專案管理指南
  - 計劃模式和子代理策略
  - 自我改進循環
  - 核心原則和最佳實踐

## 使用方法

將這些檔案複製到 `~/.claude/` 目錄（Linux/macOS）或 `C:\Users\<username>\.claude\` 目錄（Windows）。

## 同步

在任何一台電腦上更新設定後，提交並推送到這個倉庫，其他電腦就可以拉取最新配置。
