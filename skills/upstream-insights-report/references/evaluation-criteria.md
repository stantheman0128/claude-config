# 選題評估標準

## 四大評分維度（1-10 分）

| 維度 | 目標分數 | 紅旗 |
|------|---------|------|
| **Niche-ness（小眾程度）** | >8 | Bloomberg/Reuters 已有報導 |
| **Taiwan supply chain depth（台灣供應鏈深度）** | >7 | 只有 TSMC 作為通用代工角色 |
| **Timeline realism（時程現實性，1-3 年）** | >8 | <5 分 = 太早或太晚 |
| **Research difficulty（研究難度）** | >8 | 一次 Google 搜尋就能找到所有資訊 |

## 好題目的解剖學

好題目 = 以下四個元素的交集：

1. **人人知道的大趨勢**（AI、去全球化、能源轉型）
2. **幾乎沒人寫的使能子技術**（讓大趨勢成真的關鍵零組件/材料/製程）
3. **台灣供應鏈角度**（有可投資的台灣上市櫃公司）
4. **需要交叉比對非顯而易見來源才能算出的驚人數字**

## 強信號

- 如果 2+ 個獨立研究 agent 不約而同收斂到相同主題 → 信任它
- 如果題目需要讀 IEDM 論文才能理解 → niche 程度夠
- 如果你需要從 USGS 報告算出供需數學 → 研究護城河成立

## 題目類型參考

- **Picks and shovels（鎬與鏟）**：不是頭條股票，而是上游使能者
- **瓶頸材料**：某個關鍵材料的供給受限會卡住整條供應鏈
- **製程轉換**：新製程節點帶來的材料/設備需求變化
- **地緣政治供應鏈重構**：某個環節的地理集中度風險
