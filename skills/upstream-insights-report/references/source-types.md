# 非顯而易見來源類型

上游洞見報告的研究護城河在於來源品質。以下是應優先查閱的來源分類。

## 學術會議論文
- **IEDM** (International Electron Devices Meeting) — 元件層級突破
- **IITC** (International Interconnect Technology Conference) — 互連技術
- **VLSI Symposium** — VLSI 技術與電路
- **SPIE Lithography** — 微影技術進展
- **APEC** (Applied Power Electronics Conference) — 功率電子
- **OFC** (Optical Fiber Communication) — 光通訊

## 大宗商品專業報告
- **SFA Oxford** — 鉑族金屬 (PGMs) 供需分析
- **USGS MCS** (Mineral Commodity Summaries) — 礦物供應數據
- **Johnson Matthey PGM Reports** — 鉑族金屬市場年報
- **CRU Group** — 金屬與礦業分析

## 產業專業機構
- **TECHCET** — 半導體材料市場數據
- **Yole Développement** — 半導體設備與材料預測
- **LightCounting** — 光通訊市場數據
- **IC Insights / TrendForce** — 半導體產業統計

## 供應商產品頁面與白皮書
- 通常比媒體報導**領先 12+ 個月**
- 檢查新產品發布、技術規格書、應用筆記
- 特別注意產品頁面中的技術參數變化

## 專業部落格與電子報
- **BALD Engineering** — ALD 化學深度分析
- **Semiconductor Engineering** — 半導體製程技術
- **Vik's Newsletter** — 半導體產業觀察
- **SemiAnalysis** — 半導體產業深度分析

## 台灣特定來源
- **DigiTimes（中文版）** — 台灣供應鏈第一手情報
- **工商時報** — 台灣產業新聞
- **公開資訊觀測站 (MOPS)** — 上市櫃公司財報、重大訊息
- **證交所產業價值鏈資訊平台** — 產業鏈數據
- **各公司法說會簡報** — 通常在公司 IR 網站或 MOPS 可取得
